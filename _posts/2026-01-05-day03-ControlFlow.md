## 1. 오늘의 목표
코드는 기본적으로 위에서 아래로 흐릅니다. 이 흐름을 **"만약에(If)"**로 갈림길을 만들거나, **"~동안(For)"**으로 뱅글뱅글 돌게 만드는 법을 배웁니다.

---

## 2. 조건문(If): 갈림길 만들기

### 1) 만약에: `if`
* **코드:** `if condition:`
* **어원:** 고대 게르만어에서 "의심(Doubt)"이나 "조건"을 나타내는 말에서 유래.
* **설명:** 컴퓨터에게 **"일단 의심해봐. 이 조건이 맞는지 검사해봐"**라고 시키는 검문소입니다.

### 2) 아니면: `else`
* **코드:** `else:`
* **어원:** 라틴어 **Alis** (다른 것, Other) → **Else**.
* **설명:** "앞의 조건(if)이 틀렸다면, 그 밖의 **다른(Else)** 길로 가라"는 뜻입니다. 최후의 보루 같은 역할입니다.

### 3) 그게 아니고 만약에: `elif`
* **코드:** `elif condition:`
* **Full Name:** **El**se **If** (엘스 이프)의 줄임말.
* **설명:** 개발자들이 `else if`라고 두 단어 치는 게 귀찮아서 합친 약어입니다.
* **상황:**
    1. `if`: 이거니? (아니)
    2. `elif`: (**그럼 다르고**, **만약에**) 이거니? (응)
    3. `else`: 에라이 다 아니네.

### 4) 들여쓰기: `Indentation` (매우 중요)
* **어원:** **In** (안으로) + **Dens** (이빨/Teeth).
* **이미지:** 짐승이 꽉 깨물면 살이 안으로 움푹 들어가죠? 글자를 종이 안쪽으로 쑥 밀어 넣은 것을 말합니다.
* **설명:** 파이썬은 **"들여쓰기(여백)가 곧 소속"**입니다.
    * `if` 문장 아래에 **이빨 자국처럼 들여쓰기 된 코드**만 `if`의 명령을 듣습니다. 들여쓰기를 안 하면 "난 너랑 상관없는 남이야"라고 인식합니다.

---

## 3. 반복문(For): 노가다 시키기

### 1) ~를 위하여/동안: `for`
* **코드:** `for i in data:`
* **어원:** 영어 **For**는 고어에서 **"~의 앞에서"** 또는 **"~하는 동안"**이라는 뜻입니다.
* **설명:** "데이터 꾸러미 안에 있는 것들을 **하나씩 꺼내서 내 앞에 두고(For)**, 그 녀석들을 위해 명령을 반복해라"는 뜻입니다.

### 2) 범위: `range`
* **코드:** `range(0, 10)`
* **어원:** 고대 프랑스어 **Renge** (줄, 선) → 군대 용어 **Rank** (열/줄)와 뿌리가 같습니다.
* **설명:** 군인들을 연병장에 일렬로 세우듯, 숫자를 0부터 9까지 **일렬로 쫙 줄 세우는(Range)** 기능입니다. 반복문이 하나씩 집어가기 좋게 정렬해줍니다.

---

## 4. 실습 코드 (구조 파악하기)

```python
score = 75

# 1. if문 구조 (들여쓰기 주의!)
if score >= 90:
    print("A")      # 들여쓰기 O: if에 소속됨 (조건 맞아야 실행)
elif score >= 70:
    print("B")      # 들여쓰기 O: elif에 소속됨
else:
    print("C")      # 들여쓰기 O: else에 소속됨

print("종료")       # 들여쓰기 X: 위 조건들과 상관없이 무조건 실행됨 (남남)

# 2. for문 구조
# range(3)은 [0, 1, 2] 세 개의 숫자를 줄 세움
for i in range(3):
    print(i)        # 0, 1, 2가 순서대로 찍힘 (3번 반복)
```

## 🧭 오늘의 핵심 요약
1. **If** = 조건의 참/거짓에 따라 실행 여부를 결정
2. **For** = 정해진 횟수만큼 반복 수행
3. **While** = 조건이 참인 동안 무한히 반복 수행
4. **Indentation** = 파이썬에서 제어문의 범위를 결정하는 '들여쓰기'
