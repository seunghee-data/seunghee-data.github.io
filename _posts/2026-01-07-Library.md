## 1. 오늘의 목표
모든 기능을 직접 만들 필요는 없습니다. 전 세계 개발자들이 미리 만들어둔 강력한 도구들을 내 작업실로 가져오는 법을 배웁니다. **Import, Module, Library**의 어원을 통해 파이썬 생태계를 정복합니다.

---

## 2. 도구 가져오기: 수입과 별명

### 1) 수입하다: `import`
* **코드:** `import random`
* **어원:** **Im**(안으로) + **Port**(항구/나르다). 항구에서 물건을 수입하듯, 외부 코드를 내 파일 안으로 **실어 나르는** 것입니다.

### 2) 별명: `as`
* **코드:** `import pandas as pd`
* **어원:** **Alias**(별명) / **As**(~로서). "도구 이름이 기니, 내 코드 안에서는 **이 별명으로서** 부르겠다"는 선언입니다.

---

## 3. 도구의 단위: 부품과 도서관

### 1) 모듈: `module`
* **어원:** **Modulus**(작은 측정 단위). 큰 프로그램을 이루는 **작은 부품 단위(파일 하나)**를 뜻합니다.

### 2) 라이브러리: `library`
* **어원:** **Liber**(책) + **-ary**(장소). 모듈들을 주제별로 모아둔 **도서관(거대 도구함)**입니다.

---

## 4. 실습 코드 (한 줄 해석)

```python
# 1. 도구 수입 (Import)
import random           # 랜덤 도구를 수입함

# 2. 기능 사용
# random 모듈 안의 정수 추출(randint) 기능 사용
num = random.randint(1, 10) 
print(f"추출된 숫자: {num}")

# 3. 별명 붙여 사용 (As)
import datetime as dt   # 날짜 도구를 'dt'라는 별명으로 수입
now = dt.datetime.now() # 현재 시간 기능 작동
print(f"현재 시간: {now}")
```

---

## 🧭 오늘의 핵심 요약
1. **Import** = 외부 도구 수입하기 (Port: 항구)
2. **As** = 별명 지어주기 (Alias)
3. **Module** = 프로그램을 구성하는 작은 부품 (파일 단위)
4. **Library** = 부품들이 모여 있는 거대한 도구함 (도서관)
