## 2. 오늘의 핵심 용어

### 1) 출력: `print()`
* **코드:** `print("Hello")`
* **어원:** **Print** (인쇄하다)
* **설명:** 과거 컴퓨터는 결과를 종이에 인쇄했습니다. 화면에 결과를 보여주는 것을 여전히 "인쇄(Print)한다"라고 표현합니다. 즉, 컴퓨터 내부의 값을 우리 눈에 보이게 **출력**하라는 명령입니다.

### 2) 변수: `Variable`
* **코드:** `x = 10`
* **어원:** **Vary** (변하다) + **-able** (가능한) = **Variable** (변할 수 있는 것)
* **설명:** 수학의 상수(Constant, 변하지 않는 값)와 반대되는 개념입니다. 언제든 그 안의 내용물을 바꿀 수 있는 빈 상자라서 '변수'라고 부릅니다.

🧩 그럼 x = 1은 뭘 하는 걸까?

이 한 줄에는 3가지 개념이 동시에 들어 있습니다.

x = 1

요소	의미  
x	이름(Name) → 변수가 될 공간의 이름  
=	Assignment → “담아라”, “배정하라”  
1	Value → 실제 데이터  

즉, 이 문장의 진짜 의미는:

“1이라는 값을, x라는 이름의 공간에 배정하라.”

🎯 왜 코드 예시가 x = 1 인가?

사람이 Variable을 이해하려면  
이 세 가지가 동시에 보여야 합니다.

이름이 있어야 하고 → x  
값이 있어야 하고 → 1  
그 둘을 연결하는 동작이 있어야 함 → =  

그래서 가장 단순한 구조의 예시가 바로:

x = 1

이건 “x가 1과 같다”가 아닙니다.  
“x에 1을 담아라” 입니다.

수학이 아니라 메모리 조작 명령입니다.

🧭 그럼 왜 “Variable”이라는 개념이 필요할까?

만약 모든 값을 이렇게 쓴다면:

print(1)  
print(1)  
print(1)

나중에 이 1을 전부 2로 바꿔야 하면  
모든 코드를 다 찾아서 고쳐야 합니다.

하지만 이렇게 쓰면:

x = 1  
print(x)  
print(x)  
print(x)

이제 값 바꾸는 건 딱 한 줄:

x = 2

그래서 이 공간을 **Variable(변할 수 있는 것)**이라고 부릅니다.

🧱 핵심 구조 요약  
개념	정체  
Variable	이름 붙은 값 저장 공간  
=	Assignment (배정하라)  
x = 1	“x라는 공간에 1을 담아라”

이걸 이해한 순간부터  
이미 초보 단계는 넘어섰습니다.

다음으로 자연스럽게 이어지는 질문은 이거예요:

그럼 컴퓨터는 이 x를 어디에 어떻게 보관할까?

---

### 3) 할당 연산자: `=`
* **코드:** `a = 10`
* **의미:** **Assignment** (할당/배정)
* **설명:** 수학에서는 "같다(Equal)"지만, 코딩에서는 **"오른쪽의 값을 왼쪽 상자에 담는다(Assign)"**는 뜻입니다. ("같다"는 `==` 기호를 사용)

---

## 3. 자료형 (Data Type)
컴퓨터에게 데이터의 종류를 알려주는 '형태'입니다.

### ① 정수: `int`
* **코드:** `a = 5`
* **Full Name:** **Integer** (인티저)
* **어원:** 라틴어 *Integer*는 "손대지 않은", "완전한(Whole)"이라는 뜻입니다. 부스러기(소수점)가 없이 온전한 숫자, 즉 **정수**를 뜻합니다.

🧩 이 코드 `a = 5`는 무슨 뜻일까?

a = 5

요소   의미
a     이름(Name) → 메모리 공간의 이름
=     Assignment → 배정하라
5     Integer 값 → 정수 데이터

즉, 이 문장은
“정수 5를 a라는 이름의 메모리 공간에 저장하라”라는 뜻이다.

🎯 왜 하필 5일까?
정수 자료형의 가장 단순한 예시이기 때문이다.
소수점이 없고, 부호도 없고, 계산이 직관적이기 때문이다.


### ② 실수: `float`
* **코드:** `b = 3.14`
* **Full Name:** **Floating Point Number** (플로팅 포인트 넘버)
* **어원:** **Float** (둥둥 떠다니다) + **Point** (점/소수점).
* **설명:** 소수점(.)의 위치가 고정되지 않고 둥둥 떠다닐 수 있다는 뜻입니다. 컴퓨터가 소수점이 있는 숫자를 처리하는 방식이 '부동(떠다니는) 소수점' 방식이라서 `float`라고 부릅니다.

🧩 이 코드 `b = 3.14`는 무슨 뜻일까?

b = 3.14

요소   의미
b     이름(Name) → 메모리 공간의 이름
=     Assignment → 배정하라
3.14  Floating Point 값 → 실수 데이터

즉, 이 문장은
“소수점이 있는 수 3.14를 b라는 이름의 공간에 저장하라”는 뜻이다.

🎯 왜 하필 3.14일까?
원주율처럼 소수점이 있는 숫자가
정수와 구조적으로 다르다는 걸 즉시 보여주기 때문이다.


### ③ 문자열: `str`
* **코드:** `name = "Python"`
* **Full Name:** **String** (스트링)
* **어원:** **String**은 "끈", "줄"이라는 뜻입니다.
* **설명:** 글자(Character) 하나하나가 진주 목걸이처럼 실에 꿰어 줄줄이 이어져 있다고 해서 **String(문자열)**이라고 부릅니다. 따옴표(`" "`)로 묶어서 표현합니다.

🧩 왜 계속 5와 3.14를 쓰는가?

이 숫자는 ‘값’이 아니라 ‘기준점’이다.

정수 세계의 기준점 → 5  
실수 세계의 기준점 → 3.14

모든 예시는 이 기준점에서 출발해
+1, -1, ×2, ÷2, 누적, 변환, 비교 등으로 확장된다.

그래서 학습자는
“새 예제”를 외우는 것이 아니라
**같은 구조를 여러 각도에서 바라보는 경험**을 하게 된다.

## 4. 실습 코드

```python
# 1. 변수(Variable: 변하는 값을 담는 상자)에 값을 할당(Assign)
age = 25              # int (Integer: 온전한 정수)
height = 175.5        # float (Floating Point: 소수점이 떠다니는 실수)
name = "Gemini"       # str (String: 글자들이 끈처럼 이어진 문자열)

# 2. 결과 출력 (Print: 화면에 찍어내라)
print(age)
print(height)
print(name)
```

3. **String** = 따옴표로 감싸진 문자열 데이터
