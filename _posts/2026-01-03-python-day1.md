## 2. 오늘의 핵심 용어

### 1) 출력: `print()`
* **코드:** `print("Hello")`
* **어원:** **Print** (인쇄하다)
* **설명:** 과거 컴퓨터는 결과를 종이에 인쇄했습니다. 화면에 결과를 보여주는 것을 여전히 "인쇄(Print)한다"라고 표현합니다. 즉, 컴퓨터 내부의 값을 우리 눈에 보이게 **출력**하라는 명령입니다.

### 2) 변수: `Variable`
* **코드:** `x = 10`
* **어원:** **Vary** (변하다) + **-able** (가능한) = **Variable** (변할 수 있는 것)
* **설명:** 수학의 상수(Constant, 변하지 않는 값)와 반대되는 개념입니다. 언제든 그 안의 내용물을 바꿀 수 있는 빈 상자라서 '변수'라고 부릅니다.

### 3) 할당 연산자: `=`
* **코드:** `a = 10`
* **의미:** **Assignment** (할당/배정)
* **설명:** 수학에서는 "같다(Equal)"지만, 코딩에서는 **"오른쪽의 값을 왼쪽 상자에 담는다(Assign)"**는 뜻입니다. ("같다"는 `==` 기호를 사용)

## 3. 자료형 (Data Type)
컴퓨터에게 데이터의 종류를 알려주는 '형태'입니다.

### ① 정수: `int`
* **코드:** `a = 5`
* **Full Name:** **Integer** (인티저)
* **어원:** 라틴어 *Integer*는 "손대지 않은", "완전한(Whole)"이라는 뜻입니다. 부스러기(소수점)가 없이 온전한 숫자, 즉 **정수**를 뜻합니다.

### ② 실수: `float`
* **코드:** `b = 3.14`
* **Full Name:** **Floating Point Number** (플로팅 포인트 넘버)
* **어원:** **Float** (둥둥 떠다니다) + **Point** (점/소수점).
* **설명:** 소수점(.)의 위치가 고정되지 않고 둥둥 떠다닐 수 있다는 뜻입니다. 컴퓨터가 소수점이 있는 숫자를 처리하는 방식이 '부동(떠다니는) 소수점' 방식이라서 `float`라고 부릅니다.

### ③ 문자열: `str`
* **코드:** `name = "Python"`
* **Full Name:** **String** (스트링)
* **어원:** **String**은 "끈", "줄"이라는 뜻입니다.
* **설명:** 글자(Character) 하나하나가 진주 목걸이처럼 실에 꿰어 줄줄이 이어져 있다고 해서 **String(문자열)**이라고 부릅니다. 따옴표(`" "`)로 묶어서 표현합니다.

## 4. 실습 코드

```python
# 1. 변수(Variable: 변하는 값을 담는 상자)에 값을 할당(Assign)
age = 25              # int (Integer: 온전한 정수)
height = 175.5        # float (Floating Point: 소수점이 떠다니는 실수)
name = "Gemini"       # str (String: 글자들이 끈처럼 이어진 문자열)

# 2. 결과 출력 (Print: 화면에 찍어내라)
print(age)
print(height)
print(name)
