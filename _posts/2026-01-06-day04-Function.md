## 1. 오늘의 목표
매번 똑같은 코드를 복사해서 붙여넣기 하는 건 비효율적입니다. 자주 쓰는 기능을 묶어서 **"나만의 명령어(Function)"**로 만드는 법을 배웁니다. 특히 초보자가 가장 많이 틀리는 **매개변수와 인자의 차이**, **Return의 진짜 의미**를 어원으로 뜯어봅니다.

---

## 2. 함수 정의하기: 기계 설계도 만들기

### 1) 정의하다: `def`
* **코드:** `def my_function():`
* **Full Name:** **Def**ine (디파인 / 정의하다)
* **어원:** 라틴어 **De** (완전히) + **Finire** (끝내다/경계를 짓다).
* **설명:** *Finish*(끝내다)와 어원이 같습니다. "여기서부터 여기까지가 이 명령어의 기능이야"라고 명확히 **경계를 짓고 정의(Define)**한다는 뜻입니다.
* **개념:** 코드를 묶어서 이름을 붙여주는 행위입니다.

### 2) 괄호: `()`
* **의미:** **Input Slot (재료 투입구)**
* **설명:** 함수 이름 뒤에는 무조건 `()`가 붙습니다. 이 괄호는 함수라는 기계가 작동하기 위해 필요한 **재료를 받아들이는 구멍**입니다.
    * 재료가 필요하면? `(name)` 처럼 구멍을 뚫어줍니다.
    * 재료가 필요 없으면? `()` 빈 괄호로 막아둡니다.

---

## 3. 재료와 결과: 들어가는 것과 나오는 것 (가장 중요)

### 1) 매개변수: `parameter` (만들 때 기준)
* **코드:** `def hello(name):` (여기서 name)
* **어원:** 그리스어 **Para** (옆에/보조) + **Meter** (측정하다/기준).
* **설명:** 함수를 **만들 때(정의할 때)** 옆에 세워두는 **기준 변수**입니다. "이 기계는 '이름'이라는 기준이 필요해"라고 미리 자리를 잡아두는 것입니다.

### 2) 인자: `argument` (쓸 때 값)
* **코드:** `hello("Gemini")` (여기서 "Gemini")
* **어원:** 라틴어 **Arguere** (주장하다/증명하다/명확히 하다).
* **설명:** 함수를 **실제로 사용할 때(호출할 때)** 집어넣는 **구체적인 값(알맹이)**입니다. 추상적인 기준(Parameter)을 명확한 값(Argument)으로 증명해서 채워 넣는다는 뜻입니다.

### 3) 반환하다: `return` (결과물 배출)
* **코드:** `return result`
* **어원:** **Re** (다시/뒤로) + **Turn** (돌다).
* **설명:** **"제어권을 주인에게 돌려주다"**
* **동작:** 함수가 일을 다 마치고, 계산된 결과물을 들고 자기를 부른 주인(코드)에게 **되돌아가는(Return)** 것입니다.
    * **주의:** `print`는 화면에 보여주고 끝이지만, `return`은 값을 쥐어줍니다.
    * `return`을 만나는 순간 함수는 즉시 종료됩니다.

---

## 4. 실습 코드 (한 줄 해석)

```python
# 1. 함수 정의하기 (Define)
# "지금부터 'add'라는 기능을 경계 지어(Def) 만들게."
# "이 기계는 a, b라는 두 개의 기준(Parameter)이 필요해."
def add(a, b):
    total = a + b       # 들여쓰기 된 곳: 함수 내부의 일
    return total        # "계산 끝! 결과물(total)을 들고 주인에게 돌아가(Return)!"

# 2. 함수 호출하기 (Call)
# "add 기계야, 작동해!"
# "재료(Argument)로 10과 20을 증명값으로 넣어줄게."
result = add(10, 20)

# 3. 결과 확인
# add(10, 20)이 일을 마치고 '30'으로 변신해서 result에 들어감
print(result)           # 30이 출력됨
```
## 🧭 오늘의 핵심 요약
1. **Def** = 기능의 경계 짓기 (Define)
2. **Parameter** = 함수 설계도의 재료 구멍 (기준)
3. **Argument** = 구멍에 넣는 실제 알맹이 (값)
4. **Return** = 결과값 들고 주인에게 돌아가기 (Turn Back)
