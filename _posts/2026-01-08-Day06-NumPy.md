## 1. 오늘의 목표
파이썬 기본 리스트는 편리하지만 수만 개의 데이터를 처리하기엔 너무 느립니다. C언어 수준의 물리적 처리 속도를 내는 **NumPy(넘파이)**의 **수치 해석적 구조**를 분석합니다.

---

## 2. 고속 계산의 핵심: Array (배열)

### 1) 배열: `array`
* **어원:** **Ad**(~에) + **Ray**(정렬/준비). 
* **심층 분석:** 일반 리스트가 메모리 여기저기 흩어진 '바구니'라면, 배열은 데이터를 빈틈없이 일렬로 세운 **'강철 대열'**입니다. 메모리 주소가 연속적이기 때문에 CPU가 한 번에 훑고 지나갈 수 있어 압도적으로 빠릅니다.

### 2) 벡터화: `vectorization`
* **어원:** **Vector**(방향이 있는 힘) + **-ize**(화 하다).
* **심층 분석:** 반복문(for)을 쓰지 않고, 수만 개의 숫자 대열에 **동시에 계산 명령**을 내리는 방식입니다. 마치 군대에 "전원 앞으로 갓!"이라고 명령하는 것과 같습니다.

---

## 3. 데이터의 형태 분석: ndarray & Shape

### 1) 형태: `shape`
* **분석:** 데이터가 몇 층으로 쌓였는지(Dimension)를 확인하는 속성입니다. 데이터 분석에서 `shape`이 맞지 않으면 연산 자체가 불가능하므로, 데이터의 '와꾸'를 맞추는 가장 기본 지표입니다.

---

## 4. 실습 코드 (수치 해석 분석 주석)

```python
import numpy as np

# 1. 고속 대열 생성 (ndarray)
# 파이썬 리스트보다 수십 배 적은 메모리로 대용량 처리가 가능함
arr = np.array([1, 2, 3, 4, 5])

# 2. 벡터화 연산 (Vectorization)
# 내부적으로 반복문 없이 모든 원소에 동시에 10을 곱함
result = arr * 10

# 3. 데이터의 생김새(Shape) 파악
print(result.shape)
```

---

## 🧭 오늘의 핵심 분석 및 요약
1. **NumPy** = 파이썬을 수치 해석(Numerical) 전문가로 만드는 고속 엔진
2. **Continuous Memory** = 배열(Array)은 연속된 메모리 공간을 사용하여 연산 병목 현상을 해결함
3. **Vectorization** = 명시적 반복문을 제거하여 연산 효율을 극대화하는 기계 학습의 필수 테크닉
4. **Dimensionality** = 데이터 분석의 핵심은 Shape을 통해 데이터의 차원과 구조를 제어하는 것
